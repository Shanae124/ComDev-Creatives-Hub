HOW TO USE:

1. Start both servers:
   - Terminal 1: npm run backend
   - Terminal 2: npm run dev

2. Open browser:
   http://localhost:3001/admin/courses

3. Click "New Course"

4. Fill in course details → Click "Create"

5. Add modules in Tab 2

6. Upload files in Tab 3

7. All files saved to /uploads with URLs

API ENDPOINTS READY:

POST   /courses              Create course
GET    /courses              List courses
GET    /courses/:id          Get course
PUT    /courses/:id          Update course
POST   /modules              Create module
GET    /courses/:id/modules  List modules
POST   /upload               Upload file
GET    /uploads/:filename    Serve file
