## API Integration Details

### Authentication Flow
```
User Registration Form
  ↓
authAPI.register(name, email, password, role)
  ↓
Axios POST to http://localhost:3000/auth/register
  ↓
Backend validates and creates user
  ↓
Returns {token, user}
  ↓
Frontend stores token in localStorage
  ↓
Zustand store updated with user
  ↓
Redirect to dashboard
```

### Course Loading Flow
```
Dashboard component mounts
  ↓
courseAPI.getAll() called
  ↓
Axios GET to http://localhost:3000/courses
  ↓
JWT token auto-injected in headers
  ↓
Backend returns list of courses
  ↓
CourseGrid component renders courses
```

### Request/Response Flow
```
Frontend Component
  ↓
Calls API function (e.g., courseAPI.getAll())
  ↓
Axios client
  ↓
Request interceptor:
  - Add Authorization header with JWT token
  - Log request
  ↓
HTTP request to backend
  ↓
Backend processes request
  ↓
Returns response
  ↓
Response interceptor:
  - Parse JSON
  - Check for 401 errors
  - Auto-redirect to login if needed
  - Log response
  ↓
Return data to component
```

### Backend Integration
```

  ↓
Axios client (lib/api.ts)
  ↓

  ↓
JWT token injected in headers
  ↓
Express.js REST API (server.js)
  ↓
JWT middleware verifies token
  ↓
Route handler processes request
  ↓
Database query (PostgreSQL)
  ↓
Response returned
  ↓
Axios interceptor handles response
  ↓
Component receives data
  ↓
UI updates
```

## Support

### Common Issues & Solutions

**Backend won't start:**
- Check PostgreSQL running: `netstat -ano | findstr :5432`
- Check port 3000 free: `netstat -ano | findstr :3000`
- Initialize database: `node initdb.js`

**Frontend won't start:**
- Check Node.js 22+: `node --version`
- Clear cache: `npm cache clean --force`
- Reinstall: `rm -r node_modules && npm install`

**Login failing:**
- Check backend running
- Check database initialized
- Check .env.local configured
- Check browser console for errors

**Courses not loading:**
- Create test course in database
- Check Network tab for API response
- Check backend logs for errors
