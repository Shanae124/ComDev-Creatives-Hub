## 🎓 Getting Started

### For Instructors
1. Register with email, password, full name
2. Select "Instructor" role
3. Create course (title + description)
4. Add modules to organize content
5. Add lessons to modules
6. Use HTML editor to write content
7. Upload course materials (PDFs, docs, etc.)
8. Students will see your course

### For Students
1. Register with email, password, full name
2. Select "Student" role
3. Browse available courses
4. Click "Enroll" on any course
5. Access course from dashboard
6. Read lessons and download materials
7. Mark lessons as complete
8. View your progress
